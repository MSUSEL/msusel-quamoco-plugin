        public abstract Processor createProcessor(INode node)
        Constructs a processor for the provided Node. In the case that the node is a FactorNode, an Evaluator is provided, and in the case that the node is a MeasureNode then an Aggregator is provided. In all other cases a NullProcessor is provided.
        Parameters:
        node - Node a processor is required for.
        Returns:
        The newly constructed processor.
